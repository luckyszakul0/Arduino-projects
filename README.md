### Arduino projects

This repository is for small projects and other ideas I want to create while I'm in the process of learning the basics of Arduino and electronics as a whole. Each project directory contains the files with the .ino files having the main code. They also contain circuit image and component list if someone wants to reproduce the result that I've gotten.

**Project list:**
- Morse code encoder - currently very hardware-light project used for converting sentences inputted into Serial to Morse code with light and sound.
- Joystick fan - a random project that won't be further updated. It is simply a fan set on a motor with a servo to control its position. You can control the direction and speed with a joystick.
- IR controlled RGB LED - Can be used to set a mood in a room by changing the color of a LED with a remote without having to stand up.
